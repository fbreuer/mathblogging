Mathblogging.org aims to become a convenient place to check on updates on the mathematical blogosphere. We hope to implement more dynamic and personalized features as we progress.

Mimikry is the greatest compliment -- our project is a blatant ripp-off of scienceblogging.org which we admire greatly. However, the current mathematical online community is very different from the scientific one which we try to accomodate.

If you have any questions, contact mathblogging@mathblogging.org. The people behind this project in alphabetical order: Felix, Fred and Peter.
